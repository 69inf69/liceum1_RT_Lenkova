## Описание программы

Приложение «Weather Diary» предназначено для ведения дневника погоды с возможностью:

* добавления записей с указанием даты, температуры, описания погоды и наличия осадков;
* просмотра всех записей в виде таблицы;
* фильтрации записей по дате и температуре;
* сохранения данных в файл JSON и загрузки их из файла.

Программа разработана на языке Python с использованием стандартной библиотеки `tkinter` для графического интерфейса и модуля `json` для работы с данными.

## Функциональные возможности

* **Добавление записей:** ввод данных о погоде через форму с проверкой корректности.
* **Просмотр записей:** отображение всех записей или отфильтрованных данных в виде таблицы.
* **Фильтрация:**
    * по дате (отображение записей за определённый день);
    * по температуре (отображение записей с температурой выше указанного значения).
* **Сохранение и загрузка:** экспорт данных в файл `weather_data.json` и импорт из него.
* **Валидация данных:** проверка формата даты, числового значения температуры и заполнения обязательных полей.

## Технические требования

* **Язык программирования:** Python 3.6+
* **Библиотеки:** `tkinter`, `json`, `datetime` (встроенные в Python)
* **ОС:** Windows, macOS, Linux

### Примеры использования (тестовые сценарии)
## Сценарий 1: Добавление и просмотр записи

Ввести: дата — 2025-04-20, температура — 15, описание — «Переменная облачность», осадки — «Нет».
Нажать «Добавить запись».
Убедиться, что запись появилась в таблице.

## Сценарий 2: Фильтрация по температуре

Добавить несколько записей с разной температурой.
В поле «Фильтр по температуре (>) ввести 10.
Нажать «Применить фильтры».
Убедиться, что в таблице остались только записи с температурой выше +10 °C.

## Сценарий 3: Сохранение и восстановление данных

Добавить 2–3 записи.
Нажать «Сохранить в JSON».
Закрыть и перезапустить приложение.
Нажать «Загрузить из JSON».
Убедиться, что записи из файла загружены и отображаются в таблице.
